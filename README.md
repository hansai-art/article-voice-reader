# 語音朗讀器 Article Voice Reader

將任何文章轉為語音，隨時隨地聆聽學習。

🔗 **預覽網站**：[twavr.lovable.app](https://twavr.lovable.app)

## 功能特色

- 📝 **多種匯入方式**：貼上文字、上傳檔案（TXT/PDF/DOCX/MD）、輸入網址自動擷取、OCR 圖片辨識
- 🔊 **語音朗讀**：瀏覽器內建語音 + OpenAI AI 語音，可調語速
- 🤖 **AI 摘要**：一鍵生成文章重點（支援 OpenAI / Google Gemini）
- 🎵 **MP3 匯出**：將文章轉為音檔下載
- 📑 **閱讀管理**：書籤、筆記、標籤分類、閱讀進度追蹤
- 🌙 **多種閱讀主題**：預設 / 復古 / 奶油 / 深色 / 純黑
- ⚡ **速讀模式**：RSVP 一次顯示一句，專注閱讀
- 🔄 **雲端同步**：透過 Supabase 跨裝置同步
- 🌐 **中英雙語**：完整支援繁體中文與英文介面
- 📱 **PWA 支援**：可安裝至手機桌面

## 技術棧

React 18 · TypeScript · Vite · Tailwind CSS · shadcn/ui · Framer Motion · Supabase

## 開發

```bash
npm install
npm run dev
```
